## 📖 How to View

### Option 1: View Online (Recommended)
Simply visit the live portfolio link above to see the website in action.

### Option 2: Clone & Run Locally
```bash
# Clone the repository
git clone https://github.com/tarunkumardasari5725-max/Tarun-Portifolio.git

# Navigate to the project directory
cd Tarun-Portifolio

# Open index.html in your browser
# For Windows:
start index.html

# For macOS:
open index.html

# For Linux:
firefox index.html  # or any browser
```

### Option 3: Use a Local Server
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (http-server)
npx http-server

# Then visit: http://localhost:8000
```
